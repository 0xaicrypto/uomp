# UOMP × Starknet Outreach Kit

这个文件夹包含向 Starknet 社区推广 UOMP 的计划与材料。

## 文件说明

- `plan.md` — 分阶段 outreach 计划、渠道清单、衡量指标。
- `forum-post.md` — Starknet Community Forum 英文主帖草稿。
- `x-thread.md` — X/Twitter 英文 Thread 草稿。
- `one-pager.md` — 一页纸英文介绍，可用于 DM、邮件附件或会议前资料。
- `dm-templates.md` — 针对不同对象的 DM / 邮件模板。
- `grants-note.md` — Starknet 资助项目与申请思路。

## 使用顺序建议

1. 先读 `plan.md` 了解整体节奏。
2. 发布 `forum-post.md` 到 [community.starknet.io](https://community.starknet.io/)。
3. 同步发布 `x-thread.md` 到 X，@ 相关账号。
4. 用 `dm-templates.md` 联系 5–10 个目标项目。
5. 提交 [AI Portal Talk to Us](https://forms.reform.app/starkware/talk-to-us-ai-portal-starknet/3miubw) 表单。
6. 根据反馈迭代协议，并准备 Seed Grant 申请。
